# 🌸 ParisaOS ― لینوکس فمبوی خودمونی

**ParisaOS** یه توزیع لینوکس مینیمال، سبک و کاملاً شخصی‌سازیه که با عشق و کمک هوش مصنوعی (و یه فمبوی خسته اما مصمم) از صفر ساخته شده.  
اینجا خبری از شلوقی توزیع‌های بزرگ نیست. همه چیز ساده، شفاف و مال خودته.

## ✨ امکانات

- **کرنل Xen ۶.۱** با پشتیبانی از مجازی‌سازی  
  برای اینکه بتونی دنیاهای دیگه‌ای توی سیستم‌عاملت اجرا کنی.

- **میزکار JWM**  
  فوق‌العاده سبک، سریع و آماده برای تم‌های صورتی و گربه‌ای.

- **X11 کامل**  
  کلیه ابزارهای گرافیکی پایه: `xterm`، `xclock`، `xeyes` و `xinit`.

- **پکیج منیجر `opkg`**  
  مثل OpenWrt، می‌تونی بسته‌های جدید نصب کنی.

- **نصب‌کنندهٔ ساده (ISO زنده)**  
  مستقیم روی USB بریز و روی سخت‌افزار واقعی نصب کن (با احتیاط!).

- **اسم و هویت منحصربه‌فرد**  
  لوگوی خوش‌آمدگویی موقع بوت میگه: *Welcome to Femboy Linux (ParisaOS)*.
